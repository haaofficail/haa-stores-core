# G5 — Trademark Registration Checklist

> **Owner action required.** G5 of TASK-0038. **Depends on G1 (CR)**.

## What is a trademark?

The **علامة تجارية** (Trademark) protects your brand name + logo from being used by competitors. Registered at **SAIP** (Saudi Authority for Intellectual Property).

For Haa: protects "هاء متاجر" + "Haa Stores" + logo.

## Prerequisites

- [ ] **G1 CR received** (CR is the legal entity that owns the trademark)
- [ ] **Logo design finalized** (in Saudi Arabia, the logo must be filed as a separate mark or as part of the wordmark)

## Application process (6-12 months total)

### Step 1: Search SAIP database (1-2 days)

- Go to https://saip.gov.sa/
- Search for "هاء" / "Haa" / "Haa Stores" / "متاجر"
- Verify no existing trademark conflicts
- If conflicts found: consider alternative names

### Step 2: Prepare application (1 week)

Required documents:
- [ ] Trademark image (logo + wordmark)
- [ ] List of goods/services (Nice Classification — Class 35 for e-commerce, Class 9 for software)
- [ ] CR copy
- [ ] Power of attorney (if filed by agent)

### Step 3: File application (1 day)

- Online filing at SAIP
- Pay fee (~SAR 1,000 per Nice class)
- Get filing receipt + application number

### Step 4: Examination (3-6 months)

- SAIP examiner reviews the application
- May issue objections (response time: 30 days)
- Approval = publication in SAIP bulletin

### Step 5: Opposition period (2 months)

- Third parties can oppose your trademark
- If no opposition: registration granted

### Step 6: Registration granted (1-2 months after opposition)

- Trademark certificate issued (PDF + رقم العلامة)
- Valid for **10 years** (renewable)
- Update Haa platform: `tenants.trademark_number`
